# Projeto Rede Social

Projeto desenvolvido durante meus estudos de HTML e CSS, com o objetivo de praticar conceitos fundamentais de estruturação de páginas e estilização.

A proposta do projeto é simular a navegação em redes sociais dentro de uma interface inspirada em um smartphone, utilizando apenas HTML e CSS.

## Tecnologias utilizadas

- HTML5
- CSS3

## Objetivos do projeto

- Praticar a criação de layouts responsivos;
- Trabalhar com posicionamento de elementos;
- Utilizar iframes para carregamento de conteúdo;
- Aplicar estilos e efeitos visuais com CSS;
- Consolidar os conhecimentos adquiridos durante os estudos.

## Aprendizados

Durante o desenvolvimento deste projeto, pude reforçar conceitos importantes como:

- Estrutura semântica em HTML;
- Organização de arquivos e pastas;
- Uso de iframes;
- Responsividade;
- Posicionamento de elementos com CSS;
- Aplicação de imagens e estilos personalizados.
